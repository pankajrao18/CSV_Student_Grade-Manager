# CSV_Student_Grade-Manager
A simple python project to manage student grades using CSV.

## Features
- Reads student marks from a CSV file
- Calculates total, percentage and grade
- Saves results to another CSV file
- Validates marks using a custom exception
- Supports manual append of new student records.

## Concepts Used
- Python Classes
- Custom Exception Handling
- CSV File Reading/Writing
- File Append Functionality

## How To Run

1. Install Python 3
2. Run command:
   ```bash
   python grade_manager.py
